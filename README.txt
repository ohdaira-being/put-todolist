Vue.jsの学習で製作したToDoアプリです。

1,立ち上げるには。
cssファイル、htmlファイル、jsファイルの３つをpullする。
3つのファイルを同じフォルダーに入れる。
（または、html内にcssファイルとjsファイルの相対パスを書き替える。）
htmlファイルを開く。

2，仕様
・「新しいタスクの追加」
タスク名と期限を入力し、追加ボタンを押すことでタスクリストに追加することができます。
※タスク名が空欄だと追加できません。
・「タスクリスト」
追加したタスクを一覧で表示します。
・動作について
　状態・編集・削除を実装しました。
　状態：完了タスクor作業中タスクの表示を切り替えます。
完了タスクにした場合、その行が薄くなります。
　編集：押下するとタスク名と期限の編集をします。
編集が完了した時は、完了ボタンを押すことで編集を保存します。
中止したい時は、[Shift]キーを押しながらキャンセルボタンを押すことで元のデータに戻ります。
　削除：[Ctrl]キーを押しながら、削除ボタンを押すとタスクを削除します。
